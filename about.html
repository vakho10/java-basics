საიტის შესახებ

საიტზე განთავსდება პროგრამირების ენა Java-ს შესასწავლად საჭირო საბაზისო კურსის მასალა ქართულ ენაზე.

მასალა იქნება მწირი, მაგრამ სამომავლოდ გაიზრდება და გახდება (გულ)უხვი :)

ელემენტები

გაკვეთილებში არის გამოყენებული დამხმარე HTML ელემენტები, რათა ენის ათვისება გახდეს უფრო საინტერესო და გასართობი.

მოკლედ რომ ვთქვათ, ეს ელემენტები დაგეხმარებათ მასალის უკეთ სწავლაში.

ინფორმაციის მატარებელი ელემენტები

გაკვეთილებში შეხვდებით შემდეგი სამი ტიპის ინფორმაციის მატარებელ ელემენტებს: ინფორმაცია, შენიშვნა და გაფრთხილება.

ინფორმაცია

ინფორმაცია შეიცავს ისეთ ტექსტს, რომელიც უფრო ყურადსაღებია და არ გაწყენთ თუ დაიმახსოვრებთ. იგი მოთავსებულია შემდეგი სტილის ბლოკში:

ინფორმაცია რამეზე

ინფორმაციის ტექსტი რაიმე მნიშვნელოვან საკითხზე.

ბმული რაიმე მასალაზე

შენიშვნა

შენიშვნა შეიცავს ისეთ ტექსტს, რომელიც უნდა გაითვალისწინოთ კონკრებული შეცდომების თავიდან ასაცილებლად. იგი მოთავსებულია შემდეგი სტილის ბლოკში:

შენიშვნა რამეზე

შენიშვნის ტექსტი რაიმე მნიშვნელოვან საკითხზე.

გაფრთხილება

გაფრთხილება შეიცავს ისეთ ტექსტს, რომელსაც თუ არ გაითვალისწინებთ შეიძლება კოდი არ დაკომპილირდეს ან გამოუსწორებელი შეცდომა გამოიწვიოს. იგი მოთავსებულია შემდეგი სტილის ბლოკში:

გაფრთხილება ოპერაციის დამამთავრებელ სიმბოლოზე

თუ ოპერაციას (მაგალითად, ფუნქციის გამოძახებას) ბოლოში დააკებთ ; სიმბოლოს, მაშინ თქვენი კოდი არ დაკომპილირდება.

{% highlight java %}System.out.println("Hello World!") // აკლია ; ბოლოში{% endhighlight %}

კითხვა-პასუხის ელემენტი

კითხვის შემდეგ შეგხვდებათ პასუხის ჩვენების ღილაკი, რომელზე დაჭერითად გამოჩნდება პასუხის ბლოკი - პასუხის გადასამოწმებლად.
